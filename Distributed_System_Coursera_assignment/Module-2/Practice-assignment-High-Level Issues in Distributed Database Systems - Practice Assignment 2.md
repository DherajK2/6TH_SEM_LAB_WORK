## Question 2

### How does the Schema Integration Triangle address the complexities of integrating disparate database schemas?

**Answer**

- [ ] It handles user authentication and access control across integrated database systems.
- [ ] It standardizes data types and definitions across various databases.
- [ ] It enhances query performance through schema optimization.
- [x] It provides a structured approach to resolve semantic and structural conflicts between different database schemas, facilitating schema alignment and integration.
